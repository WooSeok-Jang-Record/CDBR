# 파이썬으로 파일 쓰고 읽기 
- 데이터를 기록하려면 기록할 데이터가 있어야 하는데, 여기서는 faker라는 라이브러리를 이용해서 데이터를 생성한다.
- faker는 흔히 쓰이는 필드들로 구성된 가짜 데이터를 간편하게 생성하는 수단을 제공한다.

<pre>
<code>
pip3 install faker
</code>
</pre>
